# 用户登录与权限管理

## 技术选型

| 模块 | 方案 |
|------|------|
| 认证方式 | JWT（Access Token 15min + Refresh Token 7d） |
| 密码存储 | bcrypt 哈希 |
| 邮箱发送 | nodemailer + SMTP |
| 权限模型 | 角色(admin/user) + 权限列表(chat, manage_users, manage_knowledge, system_settings) |
| 前端路由守卫 | Angular CanActivateFn |

## Task 1: 后端 - 用户实体与认证模块

**新建文件：**
- `src/auth/entities/user.entity.ts` - 用户实体（id, email, password, username, role, permissions, emailVerified, emailVerifyToken, resetPasswordToken, createdAt）
- `src/auth/dto/register.dto.ts` - 注册请求 DTO
- `src/auth/dto/login.dto.ts` - 登录请求 DTO

**User 实体关键字段：**
```typescript
@Column({ unique: true }) email: string;
@Column() password: string;  // bcrypt 哈希
@Column({ length: 50 }) username: string;
@Column({ default: 'user' }) role: string;  // 'admin' | 'user'
@Column('simple-array', { default: 'chat' }) permissions: string[];
@Column({ default: false }) emailVerified: boolean;
```

## Task 2: 后端 - JWT 认证服务

**新建文件：**
- `src/auth/auth.service.ts` - 注册/登录/验证邮箱/重置密码
- `src/auth/auth.controller.ts` - REST 接口
- `src/auth/strategies/jwt.strategy.ts` - Passport JWT 策略
- `src/auth/guards/jwt-auth.guard.ts` - JWT 守卫
- `src/auth/guards/roles.guard.ts` - 角色权限守卫
- `src/auth/auth.module.ts` - 认证模块

**API 接口：**
- `POST /auth/register` - 注册（发送验证邮件）
- `POST /auth/login` - 登录（返回 token）
- `POST /auth/refresh` - 刷新 token
- `GET /auth/verify-email?token=xxx` - 邮箱验证
- `POST /auth/forgot-password` - 发送重置密码邮件
- `POST /auth/reset-password` - 重置密码
- `GET /auth/profile` - 获取当前用户信息（需登录）

**依赖安装：** `@nestjs/jwt @nestjs/passport passport passport-jwt bcrypt class-validator class-transformer nodemailer`

## Task 4: 后端 - 权限守卫与管理员接口

**新建文件：**
- `src/auth/decorators/roles.decorator.ts` - @Roles() 装饰器
- `src/auth/decorators/permissions.decorator.ts` - @RequirePermission() 装饰器

**使用方式：**
```typescript

@Get('users')

```

**新增管理接口：**
- `GET /admin/users` - 用户列表（admin）
- `PATCH /admin/users/:id/role` - 修改用户角色（admin）
- `DELETE /admin/users/:id` - 禁用用户（admin）
